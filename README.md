# COMANDOS
## GIT BASH

1.  git config --list = comando que lista as configurações do git bash

2.  git config --global user.name = para setar o nome

3.  git config --global user.email = para setar o email

4.  git config --global user.password = para setar a senha

5.  git config --global --unset--all user.(oque vc quer tirar. Ex name, email. etc)

 ## TERMINAL

1. clear = limpa

2. cls = limpa cmd

3. cd = escolhe pasta

4. cd .. = volta pasta

5. mkdir: cria diretório

6. rmdir = apaga pasta vazia

7. tree = busca subpastas

8. edit = pode editar arquivos executáveis(tipo um texto)

9. cd/ = volta pro primeiro disco

10. explorer = abre pasta no windows

11. obs: tab = ajuda a pesquisar / autocompletar comandos 

12. git clone (link) = clona um repositório do GitHub para o PC

13. git status = clona um repositório do GitHub para o PC
    
14. git add (nome do arquivo) = adiciona o arquivo para commit

15. git commit -m(descrição) = salva as alterações com uma mensagem

16. git push (linka no github) = envia os commits para o GitHub

17. git pull ( linka do git para o pc) = atualiza o projeto do GitHub para o PC
